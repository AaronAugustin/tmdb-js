# inner searchPage(url, searchTerm, searchType, apiKey, page, includeAdult, language, additionalInfo)

Gets search data from TMDB as a JSON object.
Parameters:
Name Type Description
url string The url path.
searchTerm string The search term (query).
searchType string The type of search to perform.
apiKey string The API key to the TMDB API.
page Number The search result page to return data from.
includeAdult Boolean true if adult content will be included.
language string The language of the search results. Default value is "en-US".
additionalInfo Object Additional info to add to the search query.
